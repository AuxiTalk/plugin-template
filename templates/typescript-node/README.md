# AuxiTalk TypeScript/Node Plugin Template

> Template TypeScript/Node.js para plugins AuxiTalk.  
> TypeScript/Node.js template for AuxiTalk plugins.

## PT-BR

### Como usar

```sh
cd templates/typescript-node
npm install
npm run setup
npm test
npm run build
npm run dev
```

### O que inclui

- `plugin.json`
- runtime básico
- JSON-RPC sobre stdio
- lifecycle básico
- capability `example.ping`
- SDK helpers
- protocol harness
- testes
