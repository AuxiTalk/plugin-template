# Security

## PT-BR

- Nunca salve tokens no repositório.
- Use `.env` apenas localmente.
- Declare permissões mínimas no `plugin.json`.
- Valide entradas de capabilities.
- Não execute ações sensíveis diretamente.
- Use `action.request` para ações sensíveis.

## EN

- Never store tokens in the repository.
- Use `.env` only locally.
- Declare minimal permissions in `plugin.json`.
- Validate capability inputs.
- Do not execute sensitive actions directly.
- Use `action.request` for sensitive actions.
