# Publishing

## PT-BR

Antes de publicar um plugin:

1. atualize `plugin.json`;
2. atualize `package.json`;
3. rode `npm test`;
4. rode `npm run build`;
5. documente permissões, eventos e capabilities.

## EN

Before publishing a plugin:

1. update `plugin.json`;
2. update `package.json`;
3. run `npm test`;
4. run `npm run build`;
5. document permissions, events, and capabilities.
