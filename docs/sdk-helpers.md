# SDK Helpers

## PT-BR

O template inclui helpers simples em `src/events.ts`:

- `emitEvent(peer, event)`
- `requestAction(peer, action)`
- `queryMemory(peer, query)`
- `writeMemory(peer, entry)`
- `callAI(peer, input)`
- `createEvent(type, source, payload)`

Esses helpers encapsulam chamadas JSON-RPC comuns do plugin para o core.
