# Plugin Authoring

## PT-BR

Um plugin AuxiTalk é um processo externo que conversa com o core usando JSON-RPC 2.0 sobre stdio.

Passos para criar um plugin:

1. escolha um template em `templates/`;
2. copie para um novo repositório;
3. edite `plugin.json`;
4. implemente capabilities;
5. documente permissões e eventos;
6. rode testes;
7. publique em um repositório próprio.

## EN

An AuxiTalk plugin is an external process that communicates with the core using JSON-RPC 2.0 over stdio.

Steps to create a plugin:

1. choose a template in `templates/`;
2. copy it to a new repository;
3. edit `plugin.json`;
4. implement capabilities;
5. document permissions and events;
6. run tests;
7. publish it in its own repository.
